
# DATA6550 – AI Systems Reliability and Ethics Project

## Project Overview
This project focuses on evaluating the reliability and ethical implications of AI systems. We test different AI tools by pushing them beyond normal usage to identify errors, inconsistencies, and limitations.

---

## Objectives
- Identify failure patterns in AI systems  
- Analyze hallucinations and incorrect outputs  
- Evaluate ethical risks and biases  
- Suggest improvements for safe AI deployment  


---

## Methodology
Each team member selected an AI system and tested it using different types of questions:
- Factual questions  
- Consistency tests  
- Boundary and edge cases  
- Ethical scenarios  

All responses were recorded and analyzed.

---

## Project Structure

- **Experiments/** → Individual testing work  
- **ChatLogs/** → AI conversation logs  
- **Collaboration/** → Weekly discussions (Week A & Week B)  
- **Analysis/** → Findings and comparison  

---

## Key Focus Areas
- Reliability issues  
- Ethical concerns  
- Legal risks  
- Real-world impact  

---

## Deliverables
- Chat logs  
- Experiment results  
- Analysis and findings  
- Final report  

---

## Conclusion
This project highlights the importance of testing AI systems carefully before deploying them in real-world applications. It also emphasizes ethical responsibility and risk awareness.
